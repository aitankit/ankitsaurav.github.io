About Me

Multi-Cloud Database Engineer shaping the future of enterprise data.

As a dedicated Cloud Engineer with a deep specialization in database infrastructure, I architect, deploy, and optimize scalable data environments across Amazon Web Services (AWS), Microsoft Azure, and Google Cloud Platform (GCP). My expertise spans managed database services such as AWS RDS and Aurora, Azure SQL Database and Cosmos DB, and GCP Cloud SQL and Spanner. I excel in executing complex, zero-downtime cross-cloud migrations, implementing infrastructure-as-code (IaC), ensuring high availability, and designing robust disaster recovery pipelines. Whether it is tuning complex queries for peak performance, securing data lakes, or building multi-cloud architectures, I build the foundations that drive innovation and operational excellence.
